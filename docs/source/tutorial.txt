====================
Buildcloth Tutorials
====================

.. default-domain:: javascript

These tutorials introduce specific actions and operations using
Buildcloth tools.

:doc:`/tutorial/design-buildsystems`
   A high level introduction to build

:doc:`/tutorial/generate-makefiles`
   Describes the process of using buildcloth meta-build API to
   generate ``Makefiles``.

:doc:`/tutorial/use-buildc`
   Describes the process and format for specifying and running
   buildsystems using :doc:`buildc </api/buildc>` which provides
   wrapper around :doc:`system.py </api/system>`.

:doc:`/tutorial/custom-buildsystem`
   Introduces the possibility of using the :doc:`system.py
   </api/system>` infrastructure independently of :doc:`buildc
   </api/buildc>`.
