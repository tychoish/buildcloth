====================
Buildcloth Changelog
====================

See :doc:`/roadmap` for future development plans.

Release 0.2.0
-------------

:date: 2013-09-08

- Improved :meth:`~cloth.BuildCloth.cloth()` semantics.
- Improved :doc:`/tutorial` documentation.
- Python 3 compatibility.
- Assorted Bug fixes.
- Native-Python build system and automation, including :mod:`stages`
  and :mod:`system`.
- Command line front-end for :mod:`system`, called :ref:`buildc <dbuildc>`.
- Vast improvements to test coverage.

Release 0.1.0
-------------

:date: 2013-03-30

- Initial Release on PyPI.
- API Documentation
- Unit testing Coverage.
- :mod:`rules` module as higher level abstraction.
